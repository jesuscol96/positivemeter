How does Positivemeter work?
Go back to Positivemeter

Positivemeter is based on a convolutional neural network, which was trained using the IMDB sentiment classification dataset.

Network Architecture

The architecture is based on the covolutional model, though it implements other layers for word embedding & regularization in a sequential fashion. The model used is based on the work of Mark Omernick and Francois Chollet, view it here.

The function of each layer is described as follow:

  • Input layer: transforms text into a list of integers, where each integer is mapped to a unique word.
  • Embedding layer: replaces each integer from the previous stage into a vector of dimension 128.
  • Dropout: turns on/off each connection with the previous layer stochastically, helps with regularization.
  • Conv1D: convolutional layer, it convolves the output of the preceding layer with a kernel of size 7 and adds a bias.
  • GlobalMaxPooling: reduces complexity by taking the maximun value within a given window.
  • Dense: fully connected layer. It helps with classification of abstract features.
